# Инструкция по установке и использованию

## Плагин успешно установлен и активирован!

### Что дальше?

#### 1. Добавьте первый депозит

Перейдите в админку WordPress:
- `http://localhost:8080/wp-admin`
- В боковом меню найдите **Депозиты**
- Нажмите **Добавить новый**

#### 2. Заполните информацию о депозите

**Пример: Вклад Максад**

- **Название:** Вклад Максад
- **Изображение записи:** Загрузите логотип депозита
- **Процент в сомони:** 11
- **Процент в долларах:** 5
- **Срок:** 1 год
- **Минимальная сумма:** 1
- **Валюта:** Сомони
- **URL формы заявки:** `#deposit-form` (или оставьте пустым)
- **Контент:** Добавьте подробное описание депозита

#### 3. Создайте переводы для других языков

После публикации депозита:
- Справа увидите блок "Языки" с флажками (en, ru, tj)
- Нажмите на иконку "+" рядом с языком для создания перевода
- Заполните поля на соответствующем языке

**Пример переводов для "Вклад Максад":**

**Русский (ru):**
- Название: Вклад Максад
- Срок: 1 год

**Таджикский (tj):**
- Название: Амонати Мақсад
- Срок: 1 сол

**Английский (en):**
- Название: Maqsad Deposit
- Срок: 1 year

#### 4. Добавьте шорткод на страницу

1. Создайте или откройте страницу, где хотите отобразить депозиты
2. Добавьте блок "Шорткод" или напишите в текстовом редакторе:

```
[deposit_products]
```

3. Опубликуйте страницу

#### 5. Проверьте результат

Откройте страницу на фронтенде и вы увидите карточки депозитов!

## Примеры шорткодов

```
# Показать все депозиты
[deposit_products]

# Показать только 3 депозита
[deposit_products limit="3"]

# Показать 6 депозитов
[deposit_products limit="6"]
```

## Настройка формы заявки

В поле "URL формы заявки" можно указать:
- `#deposit-form` - якорь на странице (нужно создать форму с id="deposit-form")
- `/apply/` - относительный URL страницы
- `https://example.com/form` - абсолютный URL

## Часто задаваемые вопросы

### Как изменить цвета кнопок?

Отредактируйте файл `assets/css/deposit-products.css`:
```css
.deposit-btn-primary {
    background-color: #00008F; /* Измените на свой цвет */
}
```

### Как изменить количество колонок в сетке?

В файле `assets/css/deposit-products.css` найдите:
```css
.deposit-products-grid {
    grid-template-columns: repeat(auto-fill, minmax(320px, 1fr));
}
```

Измените на фиксированное количество колонок:
```css
.deposit-products-grid {
    grid-template-columns: repeat(3, 1fr); /* 3 колонки */
}
```

### Как добавить свои поля?

1. Откройте `deposit-products.php`
2. В методе `render_meta_box()` добавьте новое поле
3. В методе `save_meta_boxes()` добавьте сохранение поля
4. В шорткоде и шаблоне используйте новое поле

## Поддержка

Если что-то не работает:
1. Проверьте, что плагин Polylang активирован
2. Убедитесь, что в Настройки → Постоянные ссылки установлена любая структура кроме "Простые"
3. Очистите кеш WordPress
4. Проверьте консоль браузера на наличие ошибок JavaScript

## Деактивация

Если нужно отключить плагин:
1. Перейдите в Плагины
2. Найдите "Deposit Products"
3. Нажмите "Деактивировать"

**Внимание:** При удалении плагина все депозиты будут удалены!
